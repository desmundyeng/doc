MDMS (Meter Data Management System)

Meter Data Management System (MDMS) is a software platform that collects, processes, stores, and manages data generated by smart meters in utility networks.

System Architecture

Key Components

Front-end

Device Management Portal

This is where users can view real-time data from their IoT devices. It likely includes features like energy consumption readings, device status, and alerts.

Work Order Field Mobile App

This app is used by field technicians to perform work orders, such as meter installations, replacements, removal or diagnosis.

Field Diagnosis Mobile App

This app assists technicians to retrieve diagnosis information from devices in the field.

Mesh Network Visualizer

Provides a visual representation of the OpenWSN mesh network topology.

Back-end

Head End System (HES)

The central hub of the system, responsible for collecting and processing data from devices. It includes modules for data aggregation, storage, and analysis.

Meter Data Management System (MDMS)

Handles meter-specific data, such as readings, analytics, reports, work orders, customers and billing information.

Health Monitoring Services

Services that monitor the health and status of HES and portals.

Machine Learning-enabled Time-series Forecasting

Predicts future trends and patterns in consumption or other relevant data.

Machine Learning-enabled Image Recognition/Object Detection

System to identify and classify objects or patterns in images or video data.

Authentication

Authentication ensures that only authorized users can access the system. It plays a critical role in securing sensitive data and features.

Login Options

The system supports 2 login options: using Password and OTP

Login with Password is selected by default

Login with Password

Login with Password

    Users can log in using their registered email along with a secure password. When a user is created (Users Management), the user will receive an e-mail with a random secure password and link to the portal.

  1. Select Login with Password

Login with OTP

Login with OTP

    An alternative to password-based login, this option uses a One-Time Password (OTP). The OTP is sent to the user's email and must be entered to authenticate successfully. To use OTP-Based Login,

  1. Select Login with OTP

  2. Click Request OTP

  3. Check e-mail for OTP

  4. Copy and paste the 4 digit One-Time Password to OTP field and click Sign In

Change password

Change Password is only available after logging in.

  1. Tap on Profile icon

  2. Key-in current and new password. Then, select Update Password

Forgot password

  1. On login page, select Forgot Password

  2. Key-in email and select Reset Password

  3. Check email and use the new temporary password to login

Password Rules

To ensure account security, passwords must adhere to the following rules:

  • Must be at least 8 characters long.

  • Should include at least one uppercase letter (A–Z).

  • Should include at least one lowercase letter (a–z).

  • Should include at least one numeric digit (0–9).

  • Should include at least one special character (e.g., @, #, $, %).

Adding new customer

  1. Click on Add Customer

  2. Fill in Group Name and Group Description

  3. Click Create Group

    4. Only one group must be currently selected to proceed.

    Created customers will be assigned to currently selected group.

Using form

Creating customer using form

    Users can create customers one at a time using built-in form.

  1. Fill-in both Customer Information and Meter Information section

  2. Click Create Customer

    3. A unique Customer Id is mandatory. Other fields are optional and can be updated later.

Using CSV (Bulk)

Creating customer using CSV

    Users can create customers by bulk by using CSV.

  1. Click Get Add Template and wait for download to complete.

  2. Fill-in the downloaded CSV file

  3. Click Upload File and select your file

    4. A unique Customer Id is mandatory. Other fields are optional and can be updated later.

    Do not modify the file name as it contains information to create customers.

Updating customer

Updating customer information

Using form

Updating customer using form

    Users can update customers one at a time using built-in form.

  1. Update both Customer Information and Meter Information section as required

  2. Click Update Customer

    3. Customer Id cannot be modified.

    Meter Serial Number cannot be modified after an installation.

Using CSV (Bulk)

Updating customer using CSV

    Users can update customers by bulk by using CSV.

  1. Click Get Update Template and wait for download to complete.

  2. Update the downloaded CSV file as required

  3. Click Upload File and select your file

    4. Downloaded update template by default contains all customer information.

    Customer Id is permanent and should not be modified.

    Do not modify the file name as it contains information to create customers.
